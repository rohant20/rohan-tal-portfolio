# Rohan Talagadadeevi - Portfolio Website

A modern, responsive portfolio website showcasing professional experience, projects, and skills.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Smooth Animations**: Intersection Observer API for scroll-triggered animations
- **Interactive Elements**: Hover effects, smooth scrolling navigation, and animated statistics
- **Modern UI**: Clean design with gradient backgrounds and contemporary styling
- **SEO Friendly**: Semantic HTML with proper meta tags

## Sections

1. **Hero**: Eye-catching introduction with social links
2. **About**: Personal introduction with key statistics
3. **Experience**: Timeline of professional roles including Walmart internship and Recruit Athena
4. **Projects**: Showcase of 5 major projects (Recruit Athena, MLB Bet Model, Steezee, DayBy, Connexioo)
5. **Skills**: Organized display of technical skills and certifications
6. **Contact**: Contact information and social media links

## Technology Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Font Awesome Icons

## How to Use

1. Open `index.html` in any modern web browser
2. No build process or dependencies required
3. All assets are self-contained

## Customization

To customize the website:

1. **Colors**: Edit CSS custom properties in `css/styles.css` (`:root` section)
2. **Content**: Update text and links in `index.html`
3. **Images**: Add profile or project images to the `images/` folder and reference them in HTML

## Deployment

To deploy this website:

1. **GitHub Pages**: Push to a GitHub repo and enable GitHub Pages
2. **Netlify**: Drag and drop the folder to Netlify
3. **Vercel**: Connect your repo and deploy
4. **Traditional Hosting**: Upload files via FTP to any web host
